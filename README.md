# Customer Review Sentiment Analysis

## Overview
This project analyses customer reviews using NLP to classify sentiment and extract insights.

## Objectives
- Understand customer sentiment
- Identify patterns in feedback

## Tools Used
- Python
- pandas
- spaCy
- TextBlob

## Process
1. Data cleaning
2. Text preprocessing
3. Sentiment analysis
4. Insight generation

## Results
- Successfully classified reviews into sentiment categories
- Identified key themes in customer feedback

## Future Improvements
- Use advanced models (e.g. BERT)
- Improve classification accuracy
